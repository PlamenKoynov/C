Табулатор на код – табулира (слага '\t' пред всеки ред) редовете спрямо дълбочината на вложените блокове, слага нов ред
около всяка отваряща и затвяраща скоба. Прочита посочен .c файл и го форматира.

Примерен входен файл:			
int main() { if (1) {printf("blah\n");} return 0}

Примерен изходен файл:
int main()
{
if (1)
{
printf("blah\n");
}
return 0;
}
